# API Reference

This directory contains an MSDN‑style reference for all public modules in the **maf‑local** codebase. Each module file documents classes, functions, data contracts, and usage examples.

## Structure
- `modules/agents.md` – Agent classes (`LiaisonAgent`, `ProjectLeadAgent`, `DocumentationAgent`, domain leads, etc.)
- `modules/workflows.md` – Workflow implementations (`OLBWorkflow`, `ResearchWorkflow`, `MainOrchestrator`)
- `modules/tools.md` – Exported tool sets (`ALL_TIER4_TOOLS`, `ALL_TIER2_TOOLS`, `ALL_UTILITY_TOOLS`)
- `modules/models.md` – Pydantic data contracts (`TaskDefinition`, `StrategicPlan`, `ExecutorReport`)

The reference is generated manually to ensure high‑quality descriptions and examples.
