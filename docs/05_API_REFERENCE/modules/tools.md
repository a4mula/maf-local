# Tools API Reference

This document lists the public tool sets exported by the `src.tools` package.

## Tier 2 Tools
- **ALL_TIER2_TOOLS** – Strategic‑level utilities such as web search and context management.
- **ValidationTool** – Validates requirement specifications for completeness, feasibility, and conflicts. Exported from `src.tools.tier2.validation_tool`.
- Exported from `src.tools.tier2`.

## Tier 4 Tools
- **ALL_TIER4_TOOLS** – Executor‑level utilities for code execution and database access.
- Exported from `src.tools.tier4`.

## Utility Tools
- **ALL_UTILITY_TOOLS** – General purpose tools (search_web, add_context, get_context, clear_context).

## Combined Export
- **ALL_TOOLS** – Concatenation of tier‑based and utility tool lists for agent imports.
