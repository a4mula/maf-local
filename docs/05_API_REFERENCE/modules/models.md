# Models API Reference

This document lists the public data contract models used across the project.

## TaskDefinition
- **Location:** `src/models/data_contracts.py`
- **Purpose:** Represents an atomic task for executors.
- **Fields:** `description: str`, `parameters: dict`, `metadata: dict`.

## ExecutorReport
- **Location:** `src/models/data_contracts.py`
- **Purpose:** Result returned by an executor after completing a task.
- **Fields:** `task_id: str`, `status: str`, `output: Any`, `duration_ms: int`.

## PermissionFilter
- **Location:** `src/governance/permission_filter.py`
- **Purpose:** Middleware enforcing PoLA for tool execution.
- **Key Methods:** `filter(request) -> bool`.
