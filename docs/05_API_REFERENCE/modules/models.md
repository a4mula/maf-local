# Models API Reference

This document lists the public data contract models used across the project.

## TaskDefinition
- **Location:** `src/models/data_contracts.py`
- **Purpose:** Represents an atomic task for executors.
- **Fields:**
  - `task_id: str` – Unique identifier
  - `domain: str` – Domain (e.g., "Development", "QA")
  - `description: str` – Human‑readable description
  - `dependencies: List[str]` – IDs of tasks this one depends on
  - `assigned_to: Optional[str]` – Domain lead handling the task

## StrategicPlan
- **Location:** `src/models/data_contracts.py`
- **Purpose:** High‑level plan generated by `ProjectLeadAgent`.
- **Fields:**
  - `plan_id: str`
  - `target_domains: List[str]`
  - `tasks: List[TaskDefinition]`
  - `context: Optional[str]`
  - `metadata: Dict[str, Any]`
  - `created_at: datetime`

## ExecutorReport
- **Location:** `src/models/data_contracts.py`
- **Purpose:** Result returned by an executor after completing a task.
- **Fields:**
  - `executor_task_id: str`
  - `executor_name: str`
  - `status: Literal["Completed", "Failed", "Pending"]`
  - `outputs: Dict[str, Any]`
  - `error_message: Optional[str]`
  - `execution_time_ms: Optional[int]`
  - `metadata: Dict[str, Any]`

## TaskMetadata
- **Location:** `src/models/data_contracts.py`
- **Purpose:** Tracks lifecycle metadata for tasks.
- **Fields:**
  - `created_at: str` – ISO timestamp of creation
  - `updated_at: str` – ISO timestamp of last update
  - `priority: Literal["low", "medium", "high", "critical"]` (default: "medium")
  - `tags: List[str]` – Optional categorisation tags
  - `source: Optional[str]` – Origin of the task (e.g., "user", "system")

## PermissionFilter
- **Location:** `src/governance/permission_filter.py`
- **Purpose:** Middleware enforcing PoLA for tool execution.
- **Key Methods:** `filter(request) -> bool`.
