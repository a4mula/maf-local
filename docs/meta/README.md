# Meta Documentation

This folder contains documentation about the **development environment and tools** used to build MAF Local, rather than documentation about MAF Local itself.

---

- Conversation management & checkpoints

- Effective collaboration tips
- Best practices for AI pair programming

---

## Purpose

While `docs/` contains:
- Project architecture
- Implementation plans
- User guides for MAF Local
- Design rationale

`docs/meta/` contains:
- Tool usage guides (Antigravity, IDEs, etc.)
- Development workflows
- Collaboration patterns
- Environment setup tips

---

## When to Add Here

**Add to `docs/meta/` when:**
- Documenting development tools or processes
- Explaining how to work with AI assistants
- IDE configuration guides
- Git workflow patterns
- Anything meta about *how we build* vs *what we built*

**Add to `docs/` when:**
- Documenting the MAF Local project itself
- Architecture decisions
- API references
- User-facing guides
